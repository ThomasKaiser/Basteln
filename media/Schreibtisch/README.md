# Höhenverstellbarer Kinderschreibtisch

Enkel ist inzwischen im schulpflichtigen Alter also braucht es einen ergonomischen Schreibtisch:

![](001.jpg)

Herausfoderung wieder mal, möglichst viel Material zu retten statt neu zu kaufen (alles bis auf die Mahagony-Leisten an der Tischplatte für lau von [Treibgut](http://treibgut-lager.de)).

Anforderung seitens Tochter: 100x65 cm, damit es in die Ecke neben Heizung/Fenster passt (dass "neben Heizung/Fenster" ein Problem werden kann, wusste der Holzlaie - also ich – noch nicht)
